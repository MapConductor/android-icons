# CircleIcon

`CircleIcon` is a class that defines a customizable circular marker icon for use on a map.
It extends `AbstractMarkerIcon` and allows for modification of properties such as color,
size, and stroke.

The icon is rendered as a circle with a specified fill color and an optional outer stroke.
The final output is a `BitmapIcon` generated by the `toBitmapIcon()` method,
which can then be used by the map rendering system.

## Signature

```kotlin
class CircleIcon(
    private val properties: IconProperties,
) : AbstractMarkerIcon()
```

## Constructors

### CircleIcon()

Creates an instance of `CircleIcon` with customizable properties.

**Signature**

```kotlin
constructor(
    fillColor: Color = Color.Red,
    strokeColor: Color = Color.White,
    strokeWidth: Dp = Settings.Default.iconStroke,
    scale: Float = 1f,
    iconSize: Dp = Settings.Default.iconSize,
    debug: Boolean = false,
)
```

**Parameters**

- `fillColor`
    - Type: `Color`
    - Default: `Color.Red`
    - Description: The fill color of the circle.
- `strokeColor`
    - Type: `Color`
    - Default: `Color.White`
    - Description: The color of the circle's outer stroke.
- `strokeWidth`
    - Type: `Dp`
    - Default: `Settings.Default.iconStroke`
    - Description: The width of the circle's outer stroke.
- `scale`
    - Type: `Float`
    - Default: `1f`
    - Description: The scaling factor applied to the icon. A value of `1.0f` is the default size.
- `iconSize`
    - Type: `Dp`
    - Default: `Settings.Default.iconSize`
    - Description: The base size of the icon in density-independent pixels (Dp) before scaling.
- `debug`
    - Type: `Boolean`
    - Default: `false`
    - Description: If `true`, draws a bounding box and cross-lines on the icon bitmap
      for debugging alignment and sizing.

## Properties

- `fillColor`
    - Type: `Color`
    - Description: The fill color of the circle.
- `strokeColor`
    - Type: `Color`
    - Description: The color of the circle's outer stroke.
- `strokeWidth`
    - Type: `Dp`
    - Description: The width of the circle's outer stroke.
- `scale`
    - Type: `Float`
    - Description: The scaling factor applied to the icon.
- `iconSize`
    - Type: `Dp`
    - Description: The base size of the icon before scaling.
- `debug`
    - Type: `Boolean`
    - Description: A flag to enable or disable debug visuals on the icon bitmap.
- `anchor`
    - Type: `Offset`
    - Description: **Read-only.** The anchor point of the icon, specified as a fraction of
      the icon's size. The value `Offset(0.0f, 0.5f)` places the anchor at the vertical center of
      the left edge.
- `infoAnchor`
    - Type: `Offset`
    - Description: **Read-only.** The anchor point for an info window, relative to the icon.
      The value `Offset(0.5f, 0.5f)` places the info window anchor at the center of the icon.

## Methods

### copy

Creates a new `CircleIcon` instance with optionally modified properties. This is useful for
creating variations of an existing icon (e.g., for a "selected" state) without modifying
the original object.

**Signature**

```kotlin
fun copy(
    fillColor: Color = this.fillColor,
    strokeColor: Color = this.strokeColor,
    strokeWidth: Dp = this.strokeWidth,
    scale: Float = this.scale,
    iconSize: Dp = this.iconSize,
): CircleIcon
```

**Parameters**

- `fillColor`
    - Type: `Color`
    - Description: The new fill color for the copied icon.
- `strokeColor`
    - Type: `Color`
    - Description: The new stroke color for the copied icon.
- `strokeWidth`
    - Type: `Dp`
    - Description: The new stroke width for the copied icon.
- `scale`
    - Type: `Float`
    - Description: The new scaling factor for the copied icon.
- `iconSize`
    - Type: `Dp`
    - Description: The new base size for the copied icon.

**Returns**

- Type: `CircleIcon`
- Description: A new `CircleIcon` instance with the updated properties.

### copy (scale, iconSize)

A convenience overload of `copy` that updates only the `scale` and `iconSize` properties.

**Signature**

```kotlin
fun copy(
    scale: Float,
    iconSize: Dp,
): CircleIcon
```

**Parameters**

- `scale`
    - Type: `Float`
    - Description: The new scaling factor for the copied icon.
- `iconSize`
    - Type: `Dp`
    - Description: The new base size for the copied icon.

**Returns**

- Type: `CircleIcon`
- Description: A new `CircleIcon` instance with the updated `scale` and `iconSize`.

### toBitmapIcon

Generates a `BitmapIcon` from the `CircleIcon`'s properties. The resulting bitmap is
cached based on the icon's properties to improve performance, so subsequent calls
with the same properties will return the cached bitmap.

**Signature**

```kotlin
override fun toBitmapIcon(): BitmapIcon
```

**Returns**

- Type: `BitmapIcon`
- Description: The generated bitmap representation of the icon, ready to be used by a map marker.

## Example

Here is an example of how to create and use `CircleIcon`.

```kotlin
import androidx.compose.ui.graphics.Color
import androidx.compose.ui.unit.dp
import com.mapconductor.icons.CircleIcon

// 1. Create a default CircleIcon (red fill, white stroke)
val defaultIcon = CircleIcon()

// 2. Create a custom CircleIcon with specific properties
val customIcon = CircleIcon(
    fillColor = Color.Blue,
    strokeColor = Color.Yellow,
    strokeWidth = 4.dp,
    scale = 1.2f,
    iconSize = 48.dp
)

// 3. Create a variation of an existing icon using the copy() method
// This creates a larger, "selected" version of the customIcon
val selectedIcon = customIcon.copy(
    scale = 1.5f,
    strokeColor = Color.Cyan
)

// 4. Generate the BitmapIcon to be used by the map renderer
// The map framework would typically call this method internally.
val defaultBitmap = defaultIcon.toBitmapIcon()
val customBitmap = customIcon.toBitmapIcon()
val selectedBitmap = selectedIcon.toBitmapIcon()

// Now you can use these BitmapIcon instances to create markers on the map.
// e.g., map.addMarker(position, icon = customBitmap)
```
